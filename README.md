# 🎈 Luchtballon (Balloon Flight)

A side-scrolling avoidance game built with **Python** and **Pygame Zero**. Keep the balloon in the air, dodge obstacles, and beat the high scores!

## 🚀 Getting Started

### Prerequisites
Make sure you have Python installed. You will need the `pgzero` library to run the game.

### Installation
1. Clone this repository

2. Install the requirements:
```bash
pip install -r requirements.txt
```

### Running the game
Launch the game by running the main.py file:
```bash
pgzrun main.py
```

### High Scores
The game automatically tracks your performance. If you achieve a new high score, it will be saved to hoogste-scores.txt and displayed when the game ends.
